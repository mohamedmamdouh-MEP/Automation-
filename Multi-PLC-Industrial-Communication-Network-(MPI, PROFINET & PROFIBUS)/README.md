## Multi-PLC Industrial Communication Network (MPI, PROFINET & PROFIBUS)

## Overview:
Designed and implemented an industrial communication network between three Siemens S7-300 PLCs using MPI, PROFINET, and PROFIBUS communication protocols.
The project demonstrates reliable real-time data exchange between multiple PLC stations, enabling synchronized machine operation and distributed industrial automation.

The system was developed using SIMATIC Manager (STEP 7), where each PLC was programmed to exchange digital and analog data with the other controllers while maintaining stable communication and coordinated process execution.

## Features:
- Communication between three Siemens S7-300 PLCs.

- Implementation of MPI, PROFINET, and PROFIBUS networks.

- Real-time data exchange between PLC stations.

- Master-slave and peer-to-peer communication.

- Digital and analog signal transmission.

- Coordinated machine operation.

- Industrial network diagnostics and troubleshooting.

- Reliable and scalable automation architecture.

## Technologies Used:
- Siemens S7-300 PLC

- SIMATIC Manager (STEP 7)

- MPI Communication

- PROFINET

- PROFIBUS

- Ladder Logic (LAD)

## Project Architecture:

- PLC 1: Exchanges process data with PLC 2 using MPI communication.

- PLC 2: Serves as the central controller, communicating with PLC 1 via MPI and with PLC 3 via PROFINET/PROFIBUS, acting as the common communication node between both networks.

- PLC 3: Exchanges process data with PLC 2 using PROFINET/PROFIBUS communication.

This architecture demonstrates the integration of multiple industrial communication protocols within a single automation system, where PLC 2 functions as the bridge between the MPI network and the PROFINET/PROFIBUS network.
