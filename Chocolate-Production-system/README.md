## Chocolate Production Automated System

## OVERVIEW:
Designed and developed an automated chocolate production process using Siemens S7-300 PLC.
The system measures ingredient weight using a load cell, prepares recipes entered by the operator, performs automatic mixing, distributes chocolate equally into containers, and executes an automatic washing cycle after a predefined number of production batches.

## FEAUTRES:
. Recipe-based production.
. Load cell weight monitoring.
. Automatic ingredient dosing.
. Automatic mixing process.
. Equal container filling.
. Automatic cleaning cycle.
. Process monitoring and fault handling.

## TECNOLOGIES USED:
. Siemens S7-300 PLC
. SIMATIC Manager (STEP 7)
. Ladder Logic
. Load Cell
. Industrial Automation
