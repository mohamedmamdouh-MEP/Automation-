## SCADA Data Communication & Motor Drive Control

## overview:
Designed an industrial SCADA application using WinCC Flexible and SIMATIC Manager to establish communication between the PLC and HMI for monitoring and controlling a three-phase motor drive.
The project includes motor start/stop, forward/reverse operation, speed control through a Variable Frequency Drive (VFD), alarm handling, and real-time process visualization.

## features:
- PLC-HMI communication.

- Start/Stop motor control.

- Forward and reverse operation.

- Variable speed control using VFD.

- Real-time process monitoring.

- Alarm indication.

- Industrial operator interface.

## Technologies Used:
- Siemens S7-300 PLC

- SIMATIC Manager (STEP 7)

- WinCC Flexible

- VFD

- Ladder Logic
