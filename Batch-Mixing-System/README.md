# Batch Mixing and filling System

## DESCRIPTION:
developed an automated batch mixing and filling system using WinCC HMI and SIEMENS PLC S7-300.

## SOFTWARES USED:
- SIMATIC MANAGER (STEP7)
- LADDER LOGIC

 ## FEATURES:
 - Automatic mixing
 - bottle filling
 - alarm system
 - emergency stop
