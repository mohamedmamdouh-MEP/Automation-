## Automatic Transfer Switch (ATS)

## Overview:
Designed and programmed an Automatic Transfer Switch (ATS) system using Siemens PLC.
The system automatically transfers the electrical load between two transformers and two backup generators while applying safety delays to ensure continuous and reliable power supply.

## Features:
- Automatic source selection.

- Transformer failure detection.

- Generator backup operation.

- Configurable safety delay.

- Automatic load transfer.

- Fault monitoring.

- Reliable power management.

## Technologies Used:
- Siemens S7-1200 PLC

- TIA Portal V17

- Ladder Logic

## Skills Demonstrated:
- PLC Programming

- Power System Automation

- Sequential Control

- Industrial Safety
